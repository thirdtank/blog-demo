## Setup

1. Clone this repo
2. Ensure you have Docker installed
3. Build the Foundation of your dev environment:

   ```
   dx/build
   ```
4. Start it up:

   ```
   dx/start
   ```
5. In another terminal, shell into the container you just started:

   ```
   dx/exec bash
   ```
6. Run setup:

   ```
   bin/setup
   ```
7. Start the app:

   ```
   bin/dev
   ```

8. Visit `http://localhost:6502`
9. Now get to work!
